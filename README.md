# Edulith Desktop App

Official desktop application for Edulith — an AI-powered learning platform.

## Download

- Go to the [Releases](../../releases) section of this repository and download the latest version for your system
- Or download from [edulith.com](https://edulith.com) or [app.edulith.com](https://app.edulith.com) (Profile → Download Apps)

## Installation

**1. Download the correct file for your system:**

### Windows

| File | Description |
|------|-------------|
| `Edulith-Setup.exe` | Installer for Windows 10 and 11 |

Run the `.exe` file. You may see a SmartScreen warning ("Unknown publisher") — click "More info" → "Run anyway".

### macOS

| File | Description |
|------|-------------|
| `Edulith-arm64.dmg` | For Apple Silicon Macs (M1, M2, M3, M4) |
| `Edulith-x64.dmg` | For older Intel-based Macs |

Not sure which one? Click the Apple menu () → "About This Mac". If it says "Apple M1/M2/M3/M4" use the arm64 version. If it says "Intel" use the x64 version.

Open the `.dmg` and drag the app to Applications. macOS may block the app as unsigned — right-click the app → "Open" → click "Open" again to confirm. You only need to do this once.


### Linux

| File | Description |
|------|-------------|
| `Edulith.AppImage` | Universal — works on most Linux distributions, no installation needed |
| `Edulith.deb` | For Debian/Ubuntu-based distributions (Ubuntu, Linux Mint, Pop!_OS, etc.) |
| `Edulith.rpm` | For Red Hat-based distributions (Fedora, CentOS, openSUSE, etc.) |

- **AppImage** — Make executable (`chmod +x Edulith.AppImage`) and run directly. No installation required.
- **deb** — Install with `sudo dpkg -i Edulith.deb` or double-click in your file manager.
- **rpm** — Install with `sudo rpm -i Edulith.rpm` or `sudo dnf install Edulith.rpm`.

**2. Follow the on-screen setup instructions.**

## Updates

The application checks for updates automatically in the background. When an update is available, a notification appears in the title bar. The update installs the next time you restart the application.
